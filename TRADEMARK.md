about the source or quality of Nir-related software.

## Permitted Use Without Permission

You may use the Nir name and logos in the following ways without asking for
permission, as long as your use is not misleading:

- Referencing Nir by name in discussions, reviews, articles, or social media
  posts, when talking about the project.
- Using the Nir logo in a blog post or tutorial that describes how to use
  Nir, as long as the logo is not the main visual element of your own brand.
- Displaying the Nir logo in a screencast or video demonstration of the
  software.
- Using the Nir name to indicate that your project or library is compatible
  with or built for Nir (e.g., "MyLibrary for Nir"), as long as it does not
  imply official endorsement.

## Use That Requires Permission

You must obtain written permission from the trademark owner to:

- Use the Nir name or logo as part of your own product, service, or company
  name.
- Use the Nir name or logo in a domain name (e.g., nir-something.com).
- Sell merchandise (t-shirts, stickers, mugs) bearing the Nir name or logo.
- Modify the Nir logo and use the modified version for any purpose.
- Use the Nir name or logo in a way that suggests affiliation with or
  endorsement by the project.

## Community Standards

Any use of the Nir brand must not bring the project or its community into
disrepute. This includes, but is not limited to, using the brand in connection
with misleading, harmful, or unlawful activities.

## Domain Names

You may use "Nir" in a domain name or subdomain for:

- A Nir-related open source project or library.
- A community group, event, or meetup focused on Nir.
- A website or tutorial that educates others about Nir.

You may not register a domain name that could imply official status
(e.g., nir.dev, getnir.com, nir-editor.com) without permission.

## Prohibited Use

The following uses are never allowed:

- Using "Nir" or the Nir logo in a name or brand that is confusingly similar
  to the project.
- Using the Nir mark in a way that could mislead others into believing you
  are the official source of Nir or that your product is the official Nir
  distribution.
- Registering "Nir" as a trademark, company name, or domain name in any
  jurisdiction, unless you are the trademark owner.

## Trademark Notices

When using the Nir name or logo under the permitted uses above, include the
following notice in a visible location:

> "Nir and the Nir logo are trademarks of Banshal Yadav. This project is not
> affiliated with Zed Industries."

## Logo Assets

The official Nir logo files are maintained in the `assets/images/` directory
of this repository. Always use the latest versions from that directory. Do
not distort, recolor, or modify the logo except to resize it proportionally.

For questions about this policy or to request permission for a use not
